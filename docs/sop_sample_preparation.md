# 樣品製備標準作業程序 (SOP)

## 適用範圍

本 SOP 適用於銅電鍍薄膜 XRD 分析前的樣品製備與處理。

---

## 1. 基板準備

### 1.1 基板清洗
1. 使用丙酮超音波清洗 5 分鐘
2. 使用異丙醇超音波清洗 5 分鐘
3. 去離子水沖洗
4. 氮氣吹乾

### 1.2 品質要求
- 表面無可見污染
- 無明顯刮痕
- 電阻率符合規格

---

## 2. 電鍍製程

### 2.1 電鍍液配方
| 成分 | 濃度 |
|------|------|
| CuSO₄·5H₂O | 200 g/L |
| H₂SO₄ | 50 g/L |
| Cl⁻ | 50 ppm |
| 整平劑 | 0-18 mL/L |

### 2.2 電鍍條件
- 電流密度：2 A/dm²
- 溫度：25 ± 2°C
- 攪拌：150 rpm

---

## 3. XRD 測量

### 3.1 儀器設定
| 參數 | 設定值 |
|------|--------|
| X 射線源 | Cu Kα |
| 管電壓 | 40 kV |
| 管電流 | 40 mA |
| 掃描範圍 | 40° - 95° 2θ |
| 步進 | 0.02° |
| 計數時間 | 0.5 s/step |

### 3.2 樣品放置
1. 使用雙面膠固定樣品
2. 確保樣品表面與儀器平面平行
3. 調整樣品高度至標準位置

---

## 4. 資料命名規則

### 4.1 檔案格式
```
YYYYMMDD_[濃度]ml_[時間]h.txt
```

### 4.2 範例
```
20251125_0ml_2h.txt
20251126_4.5ml_24h.txt
```

---

## 5. 自退火注意事項

銅電鍍膜會發生自退火現象，需記錄：

| 記錄項目 | 說明 |
|----------|------|
| 電鍍完成時間 | 精確到分鐘 |
| XRD 測量時間 | 精確到分鐘 |
| 存放溫度 | 室溫或冷藏 |

### 5.1 時間點建議
- 0h (電鍍後立即)
- 2h, 4h, 6h, 8h
- 24h
- 48h, 72h (可選)

---

## 6. 品質控制

### 6.1 每日檢查
- [ ] X 射線管狀態正常
- [ ] 樣品台清潔
- [ ] 軟體連線正常

### 6.2 每週校準
- [ ] 使用 Si 標準樣品驗證峰位
- [ ] 檢查 FWHM 是否穩定
