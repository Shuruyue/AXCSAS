# AXCSAS 計畫書 07：微結構分析（下篇）- 缺陷與應力

**電鍍銅 XRD 分析系統 - 第七章**

---

## 1. 電鍍銅的平面缺陷

### 1.1 堆垛層錯 (Stacking Faults)

電鍍銅傾向形成大量**堆垛層錯 (SF)** 與**納米孿晶 (Nanotwins)**：

| 缺陷類型 | 形成原因 | XRD 特徵 |
|----------|----------|----------|
| 本徵堆垛層錯 | SPS 加速劑促進快速堆積 | 峰位位移 |
| 孿晶 | CEAC 機制下快速生長 | 峰形不對稱 |

### 1.2 堆垛層錯機率 α

$\alpha$ 定義為發生變形層錯的機率：
- α = 0 → 完美晶體
- α > 0 → 存在堆垛層錯

---

## 2. 峰位異常位移現象

### 2.1 Warren 理論

根據 Warren 與 Paterson 的理論，FCC 金屬中的本徵堆垛層錯導致 (111) 與 (200) 峰**相向移動**：

| 峰 | 位移方向 |
|----|----------|
| (111) | 向**高角度**偏移 ($\Delta 2\theta > 0$) |
| (200) | 向**低角度**偏移 ($\Delta 2\theta < 0$) |

### 2.2 峰間距變化

標準無缺陷銅的峰間距：
$$\Delta 2\theta_{std} = 2\theta_{(200)} - 2\theta_{(111)} = 50.433° - 43.297° = \mathbf{7.136°}$$

存在堆垛層錯時：
$$\Delta 2\theta_{exp} < 7.136°$$

---

## 3. 堆垛層錯機率計算

### 3.1 Warren 公式

$$\alpha = \frac{\Delta 2\theta_{exp} - \Delta 2\theta_{std}}{G_{111-200}}$$

其中 $G_{111-200}$ 為晶面幾何係數（約 -0.5 依 Warren 推導）

### 3.2 經驗判讀法

| 峰間距變化 | 估計 α 值 |
|------------|-----------|
| 每縮小 0.1° | α ≈ **0.5% (0.005)** |
| 縮小 0.2° | α ≈ 1% |
| 縮小 0.5° | α ≈ 2.5% |

### 3.3 計算案例

**輸入數據**：
- 實測 (111) 峰：43.38°
- 實測 (200) 峰：50.38°

**計算**：
$$\Delta 2\theta_{exp} = 50.38 - 43.38 = 7.00°$$
$$\Delta = 7.00 - 7.136 = -0.136°$$

**估計堆垛層錯機率**：
$$\alpha \approx 0.136 / 0.2 \times 1\% \approx \mathbf{0.68\%}$$

---

## 4. 堆垛層錯與添加劑關係

### 4.1 SPS (加速劑) 效應

SPS 濃度與堆垛層錯高度相關：

| SPS 濃度 | 峰間距 | α 值 | 影響 |
|----------|--------|------|------|
| 低 (< 50 ppm) | ~7.10° | < 0.2% | 正常 |
| 中 (50-100 ppm) | ~7.00° | ~0.7% | 輕微 |
| **高 (> 100 ppm)** | **< 6.90°** | **> 1%** | **電阻率上升** |

### 4.2 監測指標

| 異常狀況 | XRD 指標 |
|----------|----------|
| SPS 濃度過高 | $\Delta 2\theta_{exp} < 7.0°$ |
| 堆垛層錯累積 | 峰形不對稱（低角度側拖尾）|
| 雜質固溶 | 晶格常數擴張 |

## 5. 晶格常數異常分析

### 5.1 標準晶格常數

純銅標準晶格常數：$a_0 = 3.6150$ Å

### 5.2 電鍍銅晶格常數擴張

電鍍銅常出現 $a > 3.615$ Å：

| 原因 | 機制 | 典型擴張量 |
|------|------|------------|
| 硫 (S) 固溶 | 來自 SPS 分解 | +0.001 - 0.003 Å |
| 氯 (Cl) 固溶 | 來自鹽酸添加 | +0.001 - 0.002 Å |
| 空孔累積 | 快速沉積產生 | +0.001 Å |
| 殘餘應力 | 壓縮應力效應 | 可變 |

### 5.3 晶格常數計算

從高角度峰精修最準確：

$$a = d_{hkl} \sqrt{h^2 + k^2 + l^2}$$

推薦使用 (311) 或 (220) 峰（高角度誤差小）

### 5.4 異常判斷

| 晶格常數 | 判定 |
|----------|------|
| 3.615 ± 0.001 Å | 正常 |
| 3.616 - 3.618 Å | 輕微雜質/應力 |
| **> 3.618 Å** | **嚴重雜質固溶，需檢查** |

## 6. 室溫自退火現象

### 6.1 現象描述

電鍍銅在**室溫**下自發發生再結晶：

| 時間 | 變化 |
|------|------|
| 鍍態 (0h) | 細小晶粒 (50-100 nm)，高應力，高缺陷 |
| 數小時 | 晶粒開始長大 |
| 1-3 天 | 晶粒尺寸可達數微米 |
| 穩定態 | 結構穩定 |

### 6.2 XRD 自退火監測

| 特徵 | 鍍態 | 自退火後 |
|------|------|----------|
| (111) 峰強度 | 較弱 | **增強** |
| FWHM | 較寬 | **變窄** |
| 2θ 位置 | 可能偏移 | 接近標準 |
| 晶格常數 | 擴張 | 恢復正常 |

### 6.3 性質變化

| 性質 | 鍍態 → 自退火 | 變化幅度 |
|------|---------------|----------|
| 電阻率 | 高 → 低 | 下降 20-30% |
| 殘餘應力 | 壓縮 → 拉伸 | 反轉 |
| 硬度 | 高 → 低 | 顯著降低 |
| 晶粒尺寸 | nm → μm | 10-100× |

### 6.4 分析建議

> **重要**：進行 XRD 分析時**必須記錄樣品存放時間**
> 
> - 鍍態分析：電鍍後 < 1 小時
> - 穩定態分析：電鍍後 > 7 天

## 7. 殘餘應力分析

### 7.1 峰位位移法

殘餘應力造成晶面間距 $d$ 變化：

| 應力類型 | $d$ 變化 | 峰位位移 |
|----------|----------|----------|
| 拉伸應力 | $d$ 增大 | 向**低角度** |
| 壓縮應力 | $d$ 減小 | 向**高角度** |

### 7.2 應力計算（單軸簡化）

$$\sigma = \frac{E_{hkl}}{(1+\nu)} \cdot \frac{d - d_0}{d_0}$$

其中：
- $E_{hkl}$：方向性楊氏模數（見第 05 章）
- $\nu$：泊松比（銅約 0.34）
- $d_0$：標準間距

### 7.3 電鍍銅典型應力範圍

| 樣品狀態 | 應力類型 | 典型值 |
|----------|----------|--------|
| 鍍態（厚膜）| 壓縮 | -50 ~ -200 MPa |
| 自退火後 | 拉伸 | +50 ~ +150 MPa |
| 高添加劑 | 壓縮 | -100 ~ -300 MPa |

## 8. 綜合微結構診斷表

### 8.1 XRD 異常與原因對照

| XRD 異常 | 可能原因 | 解決建議 |
|----------|----------|----------|
| 峰間距 < 7.0° | SPS 過量 | 降低加速劑濃度 |
| $a > 3.618$ Å | 雜質固溶 | 檢查添加劑純度 |
| FWHM 隨時間變窄 | 自退火 | 記錄存放時間 |
| TC(220) > 1.5 | 高應力沉積 | 降低電流密度 |
| 峰位低角度偏移 | 拉伸應力 | 正常自退火效應 |

### 8.2 製程優化指標

| 目標 | 監測指標 | 理想範圍 |
|------|----------|----------|
| 低電阻率 | 晶粒尺寸 | > 30 nm |
| 高電遷移抗性 | TC(111) | > 1.0 |
| 良好填孔 | 峰間距 | > 7.0° |
| 低應力 | 晶格常數 | 3.615 ± 0.002 Å |
