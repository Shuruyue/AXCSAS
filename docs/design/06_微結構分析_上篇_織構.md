## 1. 織構分析概述

### 1.1 什麼是織構 (Texture)

織構是指多晶材料中晶粒取向的統計分佈：

| 類型               | 特徵               | XRD 表現         |
| ------------------ | ------------------ | ---------------- |
| **隨機取向** | 晶粒朝向無規律     | 強度比符合 JCPDS |
| **擇優取向** | 晶粒傾向某方向排列 | 強度比偏離標準   |

### 1.2 電鍍銅織構的重要性

電鍍銅的織構直接影響：

| 性質       | (111) 擇優           | (220) 擇優 |
| ---------- | -------------------- | ---------- |
| 電遷移抗性 | **高**（目標） | 低         |
| 導電性     | 較高                 | 中等       |
| 機械強度   | 中等                 | 較高       |
| 表面能     | 最低                 | 較高       |

## 2. Harris 織構係數 (TC)

### 2.1 定義公式

Harris 織構係數用於量化擇優取向程度：

$$
TC(hkl) = \frac{\frac{I(hkl)}{I_0(hkl)}}{\frac{1}{N}\sum_{i=1}^{N}\frac{I(hkl_i)}{I_0(hkl_i)}}
$$

### 2.2 參數說明

| 符號         | 意義                   |
| ------------ | ---------------------- |
| $I(hkl)$   | 實驗測得強度           |
| $I_0(hkl)$ | JCPDS 04-0836 標準強度 |
| $N$        | 考慮的峰數（通常 3-4） |

### 2.3 結果解讀

| TC 值            | 解讀                 |
| ---------------- | -------------------- |
| **TC = 1** | 隨機取向（粉末平均） |
| **TC > 1** | 此方向為擇優取向     |
| **TC < 1** | 此方向被抑制         |
| TC → N          | 極端單一取向         |

## 4. TC 計算案例

### 4.1 實驗數據

| 峰 (hkl) | $I$ (實測) | $I_0$ (標準) |
| -------- | ------------ | -------------- |
| (111)    | 15680        | 100            |
| (200)    | 5520         | 46             |
| (220)    | 4200         | 20             |

### 4.2 計算步驟

**Step 1：計算各峰正規化強度**

| 峰    | $I/I_0$         |
| ----- | ----------------- |
| (111) | 15680/100 = 156.8 |
| (200) | 5520/46 = 120.0   |
| (220) | 4200/20 = 210.0   |

**Step 2：計算平均值**

$$
\bar{X} = \frac{156.8 + 120.0 + 210.0}{3} = 162.3
$$

**Step 3：計算 TC**

| 峰    | TC =$(I/I_0)/\bar{X}$       |
| ----- | ----------------------------- |
| (111) | 156.8 / 162.3 =**0.97** |
| (200) | 120.0 / 162.3 =**0.74** |
| (220) | 210.0 / 162.3 =**1.29** |

### 4.3 結果解讀

| 峰              | TC             | 判定               |
| --------------- | -------------- | ------------------ |
| (111)           | 0.97           | 接近隨機           |
| (200)           | 0.74           | 被抑制             |
| **(220)** | **1.29** | **擇優取向** |

> 此樣品呈現 (220) 擇優取向

## 5. 快速判讀法：強度比

### 5.1 簡易判斷標準

| 比值                           | 標準值 | 判定                 |
| ------------------------------ | ------ | -------------------- |
| $I_{(200)}/I_{(111)} > 0.46$ | 0.46   | (200) 擇優           |
| $I_{(200)}/I_{(111)} < 0.46$ | 0.46   | **(111) 擇優** |
| $I_{(220)}/I_{(111)} > 0.20$ | 0.20   | (220) 擇優           |

### 5.2 電鍍銅常見模式

| 樣品類型   | 典型織構   | 強度比特徵                  |
| ---------- | ---------- | --------------------------- |
| 無添加劑   | (220) 擇優 | $I_{220}/I_{111} >> 0.20$ |
| 低電流密度 | (111) 擇優 | $I_{200}/I_{111} << 0.46$ |
| PEG + Cl⁻ | (111) 增強 | TC(111) > 1                 |
| 高 SPS     | (111) 傾向 | 伴隨堆垛層錯                |

## 9. 織構與可靠性

### 9.1 (111) 織構的優勢

(111) 擇優取向是半導體製程追求的目標：

| 優勢         | 原因                 |
| ------------ | -------------------- |
| 高電遷移抗性 | 最密堆積面，擴散困難 |
| 低表面能     | 熱穩定性好           |
| 低空洞機率   | 晶界結構緊密         |

### 9.2 織構監控指標

| 指標    | 目標範圍 |
| ------- | -------- |
| TC(111) | > 1.0    |
| TC(200) | < 1.0    |
| TC(220) | < 1.2    |
