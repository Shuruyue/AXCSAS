## 1. 電鍍銅微結構分析的核心地位

### 1.1 技術背景

在現代半導體互連技術（Interconnects）、高密度封裝（Advanced Packaging）以及印刷電路板（PCB）製造工藝中，酸性硫酸銅電鍍是構建導電線路的核心技術。

電鍍銅層的微觀結構品質直接決定：
- 電子元件的可靠性
- 導電效能
- 電遷移抗性
- 應力遷移空洞發生率

### 1.2 XRD 分析的應用價值

X 射線繞射（XRD）作為非破壞性分析手段：

| 應用面向 | 具體功能 |
|----------|----------|
| 製程監控 | 追蹤電鍍層結晶品質穩定性 |
| 機制解析 | 解讀添加劑對晶粒生長的影響 |
| 可靠度預測 | 評估電遷移抗性與應力狀態 |
| 定量分析 | 量化晶粒尺寸、織構、應變 |

## 2. 電鍍銅的微結構特殊性

### 2.1 與塊材銅的差異

電化學沈積（Electrodeposited, ED）銅膜 ≠ 冶金熔煉塊材銅

| 特徵 | 電鍍銅 | 塊材銅 |
|------|--------|--------|
| 晶粒取向 | 高度擇優取向（Texture）| 接近隨機 |
| 缺陷密度 | 高（孿晶、堆垛層錯）| 低 |
| 殘餘應力 | 複雜（添加劑共沈積）| 較低 |
| 晶粒形態 | 柱狀晶（Columnar）| 等軸晶 |
| 雜質含量 | 含 S、Cl、C 等 | 高純度 |

### 2.2 電鍍生長機制

電鍍銅層的生長受控於**電化學過電位**與**添加劑競爭吸附**：

```
陰極表面 ← 抑制劑（PEG+Cl⁻）吸附 ← 抑制電沉積
                ↑
        加速劑（SPS）置換 → 局部加速沈積
```

這種「曲率增強加速覆蓋」（CEAC）機制實現次微米溝槽無孔洞填充，但導致：
- 銅晶粒內部累積大量微觀應變
- 形成納米孿晶（Nanotwins）
- 柱狀晶垂直基板生長

### 2.3 柱狀晶生長的 XRD 意義

電鍍銅的柱狀晶特性對 XRD 分析有重要影響：

| 影響項目 | 說明 |
|----------|------|
| Scherrer K 值 | 不能使用球形假設 K=0.89 |
| 測量方向 | 柱長(生長方向) vs 柱徑(橫向) 結果不同 |
| 彈性異方性 | 必須使用方向性楊氏模數 |

## 3. 金屬銅晶體結構標準

### 3.1 面心立方 (FCC) 結構

銅結晶於面心立方（Face Centered Cubic, FCC）點陣

| 參數 | 數值 | 說明 |
|------|------|------|
| 空間群 | $Fm\bar{3}m$ | No. 225 |
| 晶格常數 $a_0$ | **3.6150 Å** | 標準值 (298K) |
| 密度 $\rho$ | 8.92 g/cm³ | |
| 堆積因子 | 0.74 | FCC 理論極限 |

### 3.2 JCPDS 04-0836 標準繞射數據

使用 Cu Kα₁ 輻射源（$\lambda = 1.54056$ Å）：

| 晶面 (hkl) | 2θ (°) | d (Å) | I/I₀ | 多重性 |
|------------|--------|-------|------|--------|
| **(111)** | **43.297** | 2.088 | **100** | 8 |
| **(200)** | **50.433** | 1.808 | 46 | 6 |
| **(220)** | **74.130** | 1.278 | 20 | 12 |
| (311) | 89.931 | 1.090 | 17 | 24 |
| (222) | 95.139 | 1.044 | 5 | 8 |

**消光規律**：FCC 晶體僅當 $h,k,l$ 全奇或全偶時產生繞射峰

### 3.3 電鍍銅的晶格常數異常

電鍍銅常出現晶格常數擴張（$a > 3.615$ Å）：

| 原因 | 機制 |
|------|------|
| 雜質原子固溶 | S（來自 SPS）、Cl、C 進入晶格間隙 |
| 空孔累積 | 高密度 Vacancies |
| 殘餘應力 | 壓縮→擴張效應 |

**實測範圍**：剛沈積銅膜可達 **3.618 Å** 以上，自退火後逐漸恢復

## 4. 電鍍添加劑分類與作用

### 4.1 三元添加劑系統

酸性硫酸銅電鍍使用三類添加劑：

| 類型 | 代表化學品 | 作用機制 |
|------|------------|----------|
| **抑制劑 (Suppressor)** | PEG + Cl⁻ | 吸附陰極表面，抑制形核，細化晶粒 |
| **加速劑 (Accelerator)** | SPS | 置換抑制劑，局部加速生長，促進孿晶 |
| **平整劑 (Leveler)** | JGB, IMEP, SH110 | 平坦化表面，細化晶粒，提升硬度 |

### 4.2 添加劑對微結構的影響預覽

| 添加劑 | 晶粒尺寸 | 織構傾向 | XRD 特徵 |
|--------|----------|----------|----------|
| 無添加劑 | 較大 | (220) | 峰窄 |
| PEG + Cl⁻ | 細化 | (111) | 峰明顯寬化 |
| SPS | 中等 | (111) | 峰間距縮小（堆垛層錯）|
| Leveler | 極細 | 隨機 | 峰極寬，硬度高 |

> **注意**：詳細的添加劑效應分析將於「06_微結構分析_上篇_織構」中展開

## 5. 樣品製備注意事項

### 5.1 電鍍樣品的 XRD 建議

| 項目 | 建議 |
|------|------|
| 樣品尺寸 | ≥ 10×10 mm |
| 表面處理 | 避免機械研磨（會引入應變）|
| 存放條件 | 記錄存放時間（自退火效應）|
| 掃描範圍 | 20° - 100° (2θ) 涵蓋主要峰 |
| 步進角度 | 0.02° - 0.05° |

### 5.2 自退火現象

電鍍銅在室溫下會發生「自退火 (Self-Annealing)」：

| 時間 | 變化 |
|------|------|
| 鍍態 (0h) | 細小晶粒（50-100 nm），高應力 |
| 數小時 | 晶粒開始長大 |
| 數天 | 晶粒可達數微米，電阻率下降 20-30% |

> **重要**：進行 XRD 分析時必須記錄**樣品存放時間**，以區分鍍態與自退火後結構
